# Project Euler Solutions - Benchmarks

Cross-language benchmark aggregation and analysis for 100 [Project Euler](https://projecteuler.net/) solutions across 11 programming languages, all generated by Claude Opus 4.6.

## What This Repo Contains

- **answers.json** -- Expected answers for correctness verification
- **collect.sh** -- Runs benchmarks across all language repos and collects results
- **aggregate.py** -- Parses benchmark output and produces comparison tables
- **JOURNEY.md** -- The full story behind this project

## Running

```bash
# Collect benchmarks from all language repos
./collect.sh

# Aggregate results into comparison tables
python3 aggregate.py
```
